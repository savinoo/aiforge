# Screenshot Guide for LinkedIn Marketing

This guide shows you how to capture professional screenshots of the AIForge chat interface for LinkedIn posts.

## Quick Start

1. **Start the dev server**:
   ```bash
   cd frontend
   npm run dev
   ```

2. **Navigate to chat**: Open `http://localhost:3000/chat`

3. **Demo mode is enabled**: No backend required! Everything works with mock data.

## Tips for Great Screenshots

### Browser Setup
1. **Disable extensions**: Use incognito/private mode for clean UI
2. **Zoom level**: 100% for crisp text
3. **Window size**: 1440x900 or larger for desktop shots
4. **Dark mode**: Toggle with system settings to show both themes

## Screenshot Tools

**macOS**:
- Cmd+Shift+4: Select area
- Cmd+Shift+4, then Space: Capture window with shadow

**Windows**:
- Win+Shift+S: Snipping tool
- Alt+PrtScn: Capture active window

**Browser Extensions**:
- GoFullPage (full-page screenshots)
- Awesome Screenshot

## Post-Processing (Optional)

1. **Add subtle shadow**: Makes screenshots pop
2. **Compress**: Use TinyPNG to reduce file size
3. **Annotations**: Add arrows/highlights for key features (use Figma, Canva, or Excalidraw)
